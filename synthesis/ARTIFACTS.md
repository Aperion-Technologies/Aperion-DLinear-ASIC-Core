## To Regenerate

```bash
cd synthesis/OpenLane

# Pull Docker image (if needed)
make pull-openlane

# Run full flow
docker run --rm \
  -v $(pwd):/openlane \
  -v $(pwd)/designs:/openlane/designs \
  -v ~/.ciel:/root/.ciel \
  -e PDK_ROOT=/root/.ciel \
  -e PDK=sky130A \
  -e PWD=/openlane \
  -w /openlane \
  ghcr.io/the-openroad-project/openlane:ff5509f65b17bfa4068d5336495ab1718987ff69 \
  ./flow.tcl -design dlinear_asic -tag new_run
```
